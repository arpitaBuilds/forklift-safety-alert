# 🚛 Forklift Safety Alert System
### Project 12 — Embedded Systems Lab

## 📋 Description
An Arduino-based safety system that uses an ultrasonic 
sensor to detect pedestrians within 2 meters of a forklift,
automatically stopping the motor and activating a buzzer alarm.

## 🔧 Components
- Arduino Uno R3
- Parallax PING Ultrasonic Sensor
- Piezo Buzzer
- DC Motor

## 📌 Pin Connections
| Component | Pin |
|-----------|-----|
| PING SIG  | 7   |
| Buzzer +  | 6   |
| Motor     | 5   |

## ▶️ Simulation
Tested on Tinkercad Circuits Simulator
